En executant (avec les droit root) le fichier speedsecure.sh via le lien ci-dessus, vous commencerez automatiquement la procédure d'installation, voici les informations a savoir :

Changement du Passeword root Installation de pwgen (Générateur de password aléatoire) Modification du passeword avec chpasswd

Changement du port SSH par défaut Génération d'un chiffre aléatoire dans une plage entre 10000 et 20000 Changement du port dans le fichier de configuration

Changement du port FTP par défaut Génération d'un chiffre aléatoire dans une plage entre 20000 et 30000 Changement du port dans le fichier de configuration

Changement du port DB par défaut (Prise en compte de différents types de DB (MariaDB / MySQL) ) Test si mysql ou mariadb Génération d'un chiffre aléatoire dans une plage entre 30000 et 40000 Changement du port dans le fichier de configuration

Désactivation de l'accès ssh pour l'utilisateur root Modification du fichier de configuration => PermitRootLogin no

Mise en place de l'accès SSH par clé Génération d'une paire de clé au format ed25519 Modification du fichier de configuration pour n'autoriser que les accès via clé

Mise en place accès VPN Installation d'OpenVPN Création d'un accès avec l'utilisateur client Génération du certificat
